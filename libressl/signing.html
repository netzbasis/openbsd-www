LibreSSL Release Signing

LibreSSL portable releases are signed with a signify(1) public key:

RWT44PcJDPu8ZDd5GfXWW2vuE+xq4M3haXXfYohnEnWoEYCKHNFut6W8

They are co-signed with GPG using this public key. (A1EB079B8D3EB92B4EBD3139663AF51BD5E4D8D5)

Release signatures can be verified using the signify program from within the OpenBSD operating system itself, using the portable version of signify, or by using GPG.
