LibreSSL Release Signing

LibreSSL portable releases are signed with a signify(1) public key: RWQg/nutTVqCUVUw8OhyHt9n51IC8mdQRd1b93dOyVrwtIXmMI+dtGFe

They are co-signed with GPG using this public key.

Release signatures can be verified using the signify program from within the OpenBSD operating system itself, using the portable version of signify, which is available with recent Debian, Ubuntu and Arch Linux releases, or by using GPG.
