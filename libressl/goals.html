LibreSSL Goals

  • Modernize the OpenSSL codebase to make it easier to audit, understand and repair.
  • Apply best-practice development processes:
    • Code Review
    • Frequent releases
    • Open development process
  • Remove obsolete or broken features and operating system support
  • Use and encourage the incorporation of secure programming interfaces in operating systems.
  • Provide secure alternatives on operating systems that do not yet have secure programming interfaces available.
