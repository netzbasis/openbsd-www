Manual pages

Web manual pages are available from OpenBSD. These manual pages reflect the latest development release of OpenBGPD.

  • bgpd - Border Gateway Protocol daemon
  • bgpd.conf - Border Gateway Protocol daemon configuration file
  • bgpctl - control the Border Gateway Protocol daemon
The BGP protocol implemented in OpenBGPD is described in RFC 4271.
