OpenBGPD Mailing Lists

General OpenBGPD user discussion list

General purpose discussions, issues and ideas may be discussed on our users@openbgpd.org list.

To subscribe, send mail to majordomo@openbgpd.org with "subscribe users" as the message body.

Most development will be done either on Github for the portable branch, or on the OpenBSD tech mailing list. Bugs should be submitted to either Github or the OpenBSD bugs mailing list.

See the OpenBSD mailing list page for more details.
