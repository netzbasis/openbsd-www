sparc64 PLT Update Example

  • target address within 2^31 of PLT slot

      .PLTn: sethi  (. - .PLT0), %g1      # load offset of entry
             mov    %o7, %g1              # save return address in %g1
             call   (addr - .)            # relative call
              mov   %g1, %o7              # restore return address
             nop; nop
             nop; nop
