OpenBSD - The Principles
  • Project goals
    • Full access to the source code for everyone
    • Good code with a free license, GPL only as an exception
    • Proactive security and the goal to have the most secure OS
  • Policies
    • OpenBSD is free without any non-free stuff...
    • ...or even something "less free"
    • Have simple copyrights like the traditional "Berkeley" ones
    • Focus on portability, correctness and proactive security
