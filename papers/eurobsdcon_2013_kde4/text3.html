KDE4 on OpenBSD: CMake

  • Support for OpenBSD shared libraries policy:

    • software from ports should install shared libraries under $PREFIX/lib
    • $PREFIX is almost always /usr/local
    • libfoo.so.0.0 file under lib/ only; no libfoo.so.0 or libfoo.so symlinks
