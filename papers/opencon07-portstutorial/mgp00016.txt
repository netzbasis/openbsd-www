
Anatomy of a port (cont.)

Makefile
The Makefile contains all the information on how to build the port.

distinfo
This file records hashes and the size of the distfile(s).

patches/
Patches against the source code of an application are stored in this directory.

pkg/DESCR
This file contains a brief description of the application.

pkg/PLIST
This is the packaging list, which contains instructions for package creation once the application has been built.
