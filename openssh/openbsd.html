OpenSSH for OpenBSD

Since OpenSSH is developed by the OpenBSD group, OpenSSH is included in the base operating system, starting at OpenBSD release 2.6. So there's nothing to install or add to your system. However, due to newer features or other patches, you may wish to upgrade.

You can download the current OpenSSH release from these mirrors.

The following versions of OpenSSH are available for putting on your OpenBSD machines:

  • OpenSSH 7.2: February 29, 2016

    http://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-7.2.tar.gz

    If you are installing OpenSSH 7.2 on OpenBSD 5.7 or 5.8, you will need the the following patch:
    http://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd58_7.2.patch. 

    # cd /usr/src/usr.bin
# tar xvsfz /openssh-7.2/ssh/ .../openssh-7.2.tar.gz
# cd ssh
# patch -p0 < /path/to/openbsd58_7.2.patch # if on OpenBSD 5.7 or 5.8
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
    and restart sshd.
