OpenNTPD Manual Pages

Web manual pages are available from OpenBSD. These manual pages reflect the latest development release of OpenNTPD.

  • ntpd — Network Time Protocol daemon
  • ntpctl — Network Time Protocol query program
  • ntpd.conf — Network Time Protocol daemon configuration file

The NTP protocol implemented in OpenNTPD is described in RFC 1305 and RFC 5905.

A reference implementation and other documentation of the NTP protocol is available from ntp.org.
