OpenIKED Manual pages

Web manual pages are available from OpenBSD. These manual pages reflect the latest development release of OpenIKED.

  • iked — Internet Key Exchange version 2 (IKEv2) daemon
  • iked.conf — IKEv2 configuration file
  • ikectl — control the IKEv2 daemon

The IKEv2 protocol implemented in OpenIKED is described in RFC 5996.
